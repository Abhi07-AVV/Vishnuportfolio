02 / ML CLASSIFIER
Helpdesk Sentiment Analysis System
Python NLTK Scikit-Learn Gradio Speech-to-Text
GitHub ↗
Challenge
Helpdesk teams lack automated tooling to understand the emotional tone of customer call transcripts at scale — leaving sentiment blind spots that affect service quality decisions.
Action
Processed 900+ audio-derived transcripts using NLTK tokenization and TF-IDF vectorization with optimized n-gram filters. Benchmarked four ML models — Logistic Regression, Naive Bayes, Linear SVC, and Random Forest. Integrated Google Speech-to-Text for raw audio ingestion and Gradio for a real-time web interface.
Result
Achieved 95% classification accuracy with Logistic Regression, outperforming SVM (93%) and Random Forest (94%). Delivered a live, end-to-end audio-to-sentiment pipeline.
🎯 95% model accuracy
📄 900+ transcripts processed
